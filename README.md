# MSI-Field-Directory

This repository contains the source code for the MSI Field Directory website, a public-facing resource that displays approved and published information about soccer fields used by MSI.

The website provides:
- A searchable table of fields
- Field characteristics and amenities
- Address-based map links
- Responsive design for desktop and mobile

Built with:
- Static HTML, CSS, and JavaScript
- JSON-based data source
- GitHub Pages for hosting

To see the website, visit https://shelbyroman1.github.io/MSI-Field-Directory/.

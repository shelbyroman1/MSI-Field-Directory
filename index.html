MSI Field Directory

This directory provides coaches, players, and families with helpful details about MSI practice and game fields. Use it to understand field size, surface type, parking availability, and amenities before arriving.

  • Use the Search box to find a field by name, address, or other attribute.
  • Click any column header to sort.
  • Use the entries per page dropdown to change how many rows are shown.
  • On a cell phone or other mobile device, landscape mode may be easier to see the wide table.

  • Surface Type: Type of grass or turf on the field (Turf / Bermuda grass / Standard grass / Weeds/patches / Mostly dirt).
  • Suitable Game Size: Typical match format the field supports (7v7 / 9v9 / 11v11).
  • Parking Capacity: General estimate of the number of parking spaces available at the facility (Small / Medium / Large).
    • Small = 0-20 parking spaces.
    • Medium = 20-50 parking spaces.
    • Large = 50+ parking spaces.
  • Parking Lot Location: General setting of available parking near the field (Residential neighborhood / Commercial/Mixed-Use area / Near major road/highway).
  • Seating: Bleachers / Benches / None available.
  • Restrooms: On-site restroom availability (Permanent restrooms / Temporary restrooms, such as PortaPotties / None available).
  • Playgrounds: Whether there is a playground at the facility or nearby. (Yes / No).
